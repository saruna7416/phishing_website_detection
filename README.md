This project detects phishing websites using machine learning algorithms. 
Phishing attacks are fraudulent attempts to steal sensitive information 
such as usernames, passwords, and credit card details by creating fake 
websites that imitate legitimate ones.
The system analyzes website and URL features to classify whether a site 
is legitimate or phishing, helping improve cybersecurity and protect users 
from online fraud.

## Objectives
- Detect phishing websites using machine learning
- Improve cybersecurity awareness
- Classify websites as phishing or legitimate
- Compare multiple machine learning algorithms

## Machine Learning Algorithms
- Logistic Regression
- Random Forest
- Support Vector Machine (SVM)
- Neural Networks
- CART (Classification and Regression Trees)

## Dataset
The dataset contains phishing and legitimate website records.
Features include URL characteristics, domain information, 
and webpage attributes used to train machine learning models.
